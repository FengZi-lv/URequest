# URequest v1.0

基于python3编写的网络请求工具。

虽然可能没有fiddler之类的软件好用，但。。。。

使用方法：

```powershell
python3 urequest.py
```

# English Document

Network request tool written based on Python 3.

Maybe not as good as Fiddler or something, but...

Usage:

```powershell
python3 urequest.py
```
